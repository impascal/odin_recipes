# Odin Recipes
A simple recipe webpage for the Recipes project in The Odin Project (foundations).
Simple HTML pages with links, images, lists etc. The images are not mine but they're
all free for use.

I decided to use the "Big 4s" of Italian/Roman pasta recipes as content for the website.
You will find recipes for:
* _Amatriciana_;
* _Carbonara_;
* _Cacio e Pepe_;
* _Gricia_;
